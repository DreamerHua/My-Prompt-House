# My Prompt House - 个人提示词收藏管理系统

## 项目背景

在日常工作和学习中，我经常需要使用各种提示词（Prompt）来与AI模型交互，以获取所需的信息或生成内容。然而，随着提示词的管理往往成为一个痛点：

- **分散存储**：提示词通常散落在各个文档、笔记或编辑器中，难以集中管理
- **缺乏标签复用**：好的提示词难以以快速查找和复用，需要重复输入或复制粘贴
- **缺乏组织**：没有有效的分类和标签系统，难以按主题或用途整理提示词
- **效率低下**：在需要时难以快速找到合适的提示词，影响工作效率

My Prompt House 正是为了解决决这些痛点而设计的解决方案，帮助您高效管理和复用用以往提示词。

## 核心功能

### 1. 提示词卡片管理
- 创建、编辑和删除提示词卡片
- 每张卡片包含标题、内容和标签
- 卡片支持网格视图和列表视图切换

### 2. 标签系统
- 为提示词添加自定义标签
- 通过标签快速筛选和查找相关提示词
- 支持多标签组合筛选

### 3. 本地数据存储
- 使用浏览器localStorage存储数据
- 无需后端服务器，直接在浏览器中运行
- 数据持久化，页面刷新后不丢失

### 4. 用户友好界面
- 简洁美观的UI设计
- 流畅的动画和过渡效果
- 响应式布局，适配各种设备

## 如何使用

### 本地运行
1. 克隆取或下载本项目到本地
2. 直接在浏览器中打开 `index.html` 文件
3. 无需安装任何依赖，项目使用CDN加载所需资源

### 基本操作

#### 创建新提示词
1. 点击顶部的"新建提示词"按钮
2. 在弹出的表单中输入标题、内容和标签
3. 点击"保存"按钮创建新的提示词卡片

#### 管理标签
1. 在创建或编辑提示词时，可以输入新标签或选择已有标签
2. 点击标签筛选区的标签可以快速筛选相关提示词
3. 支持同时选标签同时筛选

#### 编辑和删除
1. 点击卡片上的编辑按钮可以编辑提示词
2. 点击删除按钮可以删除不需要的提示词（删除前会有确认提示）

#### 复制内容
1. 点击卡片上的"复制"按钮可以快速复制提示词内容到剪贴板

## 数据安全

- 所有数据仅存储在您的浏览器本地，不会上传到任何服务器
- 清除浏览器数据可能会导致提示词丢失，请谨慎操作
- 建议定期导出重要提示词，以防数据丢失

## 技术实现

- HTML5 + CSS3 + JavaScript
- Tailwind CSS 用于样式设计
- Font Awesome 提供图标支持
- localStorage 用于本地数据存储
